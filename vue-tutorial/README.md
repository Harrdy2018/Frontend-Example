## 创建Vue3.0工程 ```https://cli.vuejs.org/zh/guide/```
### 使用vue-cli创建
* vue --version
* npm install -g @vue/cli
* vue create vue3_test
* npm run serve
### 使用vite创建 ```https://vitejs.cn/```
* ```https://v3.cn.vuejs.org/guide/installation.html#vite```
* npm init vite-app vue3_test_vite
* cd vue3_test_vite
* npm install
* npm run dev
